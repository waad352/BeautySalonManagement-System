# Elegance Beauty Salon Management System

A desktop application built with **Java** and **JavaFX** that simulates a beauty salon's booking and billing system.

## ✨ Features
- **Interactive GUI:** Created using JavaFX to handle user selections.
- **Service Management:** Includes services for Hair, Makeup, Nails, and Skin Care.
- **OOP Implementation:** Demonstrates the use of Inheritance, Abstraction, and Interfaces.
- **Dynamic Billing:** Calculates total costs and processes payments (Cash/Card).

## 🛠️ Technologies
- Java SDK
- JavaFX Library
